Інтерактивні вправи та алгоритми створення

XMind – Ментальна карта «Основні складові ПК»

Мета: Систематизація знань про архітектуру комп’ютера та візуалізація зв’язків між компонентами.

Алгоритм створення:

  1. Відкрити XMind (онлайн або десктопну версію).
  2. Створити нову карту. Центральна тема: «Персональний комп’ютер».
  3. Додати гілки першого рівня: Системний блок, Периферійні пристрої, Додаткові пристрої.
  4. Додати підкатегорії: наприклад, Системний блок → Процесор, Материнська плата, Відеокарта.
  5. Застосувати кольорове кодування гілок і додати іконки (Insert → Icon).
  6. Переглянути карту у режимі Presentation Mode.
  7. Поділитися карткою: Share → Link або Export → PNG/PDF.

LearningApps – «Пристрої введення та виведення»

Мета: Відпрацювання навичок класифікації пристроїв ПК.

Алгоритм створення:

  1. Увійти на LearningApps.org.
  2. Обрати шаблон: Create App → Group Assignment (Сортування картинок).
  3. Створити дві групи: Введення та Виведення.
  4. Додати картинки пристроїв (клавіатура, миша, монітор, принтер, сканер) через Add Media → Upload Image.
  5. Призначити кожну картинку до групи (Assign → Input / Output).
  6. Налаштувати повідомлення про успіх / помилку (Settings → Feedback → Correct / Incorrect).
  7. Зберегти вправу: Save → Share → Copy Link.

Genially – Інтерактивний плакат «Види комп’ютерів»

Мета: Ознайомлення з різновидами сучасної обчислювальної техніки.

Алгоритм створення:

  1. Авторизуватися на Genially.
  2. Create → Interactive Image.
  3. Завантажити фон: колаж із різних типів комп’ютерів.
  4. Додати Hotspots на кожен об’єкт (Add Interaction → Popup).
  5. Заповнити спливаючі вікна описом та цікавими фактами.
  6. Додати анімації для Entrance/Exit у спливаючих вікнах.
  7. Переглянути в режимі Preview → Share → Copy Link.

Перейти до вправи →

Tiki-Toki – Стрічка часу «Історія комп’ютерної техніки»

Мета: Формування уявлення про хронологію розвитку ЕОМ.

Алгоритм створення:

  1. Зареєструватися на Tiki-Toki.
  2. Create → New Timeline.
  3. Встановити часовий діапазон: 1642р. - 2025р.
  4. Додати події для кожного покоління ЕОМ (I – лампи, II – транзистори тощо).
  5. Додати фото та опис для кожної події (Create new story → Title / Date / Intro / Media / Extra info).
  6. Додати мультимедіа: відео, зображення для інтерактивності.
  7. Preview → Share → Copy Link.
